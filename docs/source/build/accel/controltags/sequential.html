SequentialÂ¶

Sequential is a controller tag which tells the compiler not to attempt to parallelize or to pipeline inner computation. In this scheduling mode, the controller’s counter will only increment when it’s last stage is complete. This tag is needed primarily when your algorithm contains long loop-carry dependencies that cannot be optimized away.
