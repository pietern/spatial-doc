Pipe
====

**Pipe** is the default controller tag in Spatial, and doesn't usually need to be specified.
This tag tells the compiler that the stages of the controller can be overlapped in a pipelined fashion.
If the controller contains other controllers within it, this means that these inner controllers will be executed using
coarse-grained pipeline scheduling.
Additionally, Pipe tells the compiler it may attempt to parallelize the loop by unrolling it in space.
